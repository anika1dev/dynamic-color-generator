# Dynamic Color Generator

A simple and interactive color generator built using HTML, CSS, and JavaScript.  
Click the button to generate random background and button colors dynamically in real time.

## Features

- Random background color generation
- Dynamic button color changes
- Responsive and clean UI
- Smooth color transition effects
- Built with vanilla JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript
